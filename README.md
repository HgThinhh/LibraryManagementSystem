# LibraryManagementSystem

## Giới thiệu

Phần mềm quản lý thư viện được phát triển bằng C# WinForms và SQL Server nhằm hỗ trợ quản lý sách, độc giả, kho sách và quá trình mượn trả trong thư viện.

## Công nghệ sử dụng

* C#
* Windows Forms
* SQL Server
* LINQ to SQL
* Crystal Reports
* Mô hình 3 lớp (3-Tier Architecture)

## Chức năng chính

### Quản lý sách

* Thêm sách
* Sửa sách
* Xóa sách
* Tìm kiếm sách

### Quản lý độc giả

* Quản lý thông tin độc giả
* Quản lý thẻ thư viện

### Quản lý mượn trả

* Lập phiếu mượn
* Lập phiếu trả
* Theo dõi quá hạn

### Quản lý kho

* Quản lý vị trí sách
* Quản lý kệ sách

### Báo cáo thống kê

* Thống kê phiếu mượn
* Thống kê phiếu phạt
* Thống kê sách theo tác giả
* Thống kê sách theo thể loại
* Xuất báo cáo bằng Crystal Reports
